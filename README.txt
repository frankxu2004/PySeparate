Readme

The program consists of 3 parts. In the /console folder, there is a word segmentation program under console (Text to separate is in the deal.txt). In the /dist folder, there is a distribution of excetubles of the program (No Python Runtime pre-installation needed). In the /src folder, there is the source code in python file. 

Note: 1. In order to run the source file, you have to have Python 3.2.3 installed in the computer.
      2. The directory of the file to import should not have Chinese characters due to the python codecs problem.
